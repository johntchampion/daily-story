# Daily Story

Visit this page every day to see a story in the language you're learning. Select the language and skill level, and a story will be available for you at the skill level you specify.

## Running On Your Machine

Node.js must be installed on your machine. Install dependencies and then run the app.

```
npm install
npm run dev
```

In production, build the project and run from the dist folder.

```
npm install
npm run build
npm start
```
